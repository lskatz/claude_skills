# claude_skills

A collection of reusable [Claude](https://claude.ai) skill files.

## Folder Structure

```
claude_skills/
├── skills/              # All skill files, organized by category
│   ├── coding/          # Programming, development, and debugging
│   ├── writing/         # Writing, editing, and documentation
│   ├── analysis/        # Data analysis, research, and reasoning
│   ├── automation/      # Task automation and workflows
│   └── general/         # General-purpose and miscellaneous
├── templates/           # Templates for creating new skills
│   └── skill_template.md
├── LICENSE
└── README.md
```

## Usage

1. Browse the `skills/` directory to find a skill you need.
2. Copy the contents of the skill's `.md` file.
3. Paste it into Claude as a skill or system prompt.

## Creating a New Skill

1. Copy `templates/skill_template.md` as a starting point.
2. Fill in the description, instructions, and examples.
3. Save it in the appropriate category under `skills/`.
