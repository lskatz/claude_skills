# Skills

This directory contains Claude skill files organized by category.

Each subdirectory represents a category of skills. Skill files are written in Markdown (`.md`).

## Categories

| Directory      | Description                                      |
| -------------- | ------------------------------------------------ |
| `coding/`      | Programming, development, and debugging skills   |
| `writing/`     | Writing, editing, and documentation skills        |
| `analysis/`    | Data analysis, research, and reasoning skills     |
| `automation/`  | Task automation and workflow skills               |
| `general/`     | General-purpose and miscellaneous skills          |

Feel free to add new category directories as needed.
